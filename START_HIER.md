# START HIER – GridAtlas Website austauschen

1. Sichere bei GitHub den aktuellen Stand als ZIP.
2. Lösche im Repository die alten Website-Dateien, aber nicht das Repository selbst.
3. Lade **den Inhalt dieses Ordners direkt in die Repository-Root** hoch.
4. Prüfe, dass `index.html`, `CNAME`, `assets/` und `pv-netzanschluss-check/` direkt sichtbar sind.
5. Commit-Nachricht: `Launch GridAtlas self-check and organic growth website`.
6. Warte auf GitHub Pages und teste `https://gridatlas.de` im privaten Browserfenster.
7. Teste den Selbstcheck und sende eine Testanfrage über Formspree.
8. Reiche `https://gridatlas.de/sitemap.xml` bei Google Search Console und Bing Webmaster Tools ein.

**Nicht öffentlich hochladen:** Product Core, Wissens-OS, Kundendokumente, `.env`, Zugangsdaten oder n8n-Konfigurationen.
