# GridAtlas Website

Offizielle öffentliche Website von **GridAtlas**:

- Website: https://gridatlas.de
- Produktdemo: https://gridatlas.de/demo/
- Kostenloser Prozesscheck: https://gridatlas.de/pv-netzanschluss-check/

GridAtlas ist ein dokumentenbasierter PV-Preflight für Elektro- und PV-Betriebe in Deutschland. Die Website befindet sich in begrenzter Markterprobung und verarbeitet im öffentlichen Prozesscheck keine Kundendokumente.

## Technischer Aufbau

- statisches HTML, CSS und Vanilla JavaScript
- keine Datenbank und kein Produktlogin auf der Marketingdomain
- Formspree für freiwillig abgesendete Formulare
- keine Drittanbieter-Tracker
- GitHub Pages als derzeitige statische Veröffentlichung
